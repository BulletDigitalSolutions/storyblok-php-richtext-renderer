# Storyblok PHP Richtext Renderer
The utility class for renderer HTML from Richtext component in Storyblok.

### Install dependecies

```shell
  composer install
```

#### Testing
We use phpunit module for tests. In terminal, execute:
```bash 
  composer run test
```
